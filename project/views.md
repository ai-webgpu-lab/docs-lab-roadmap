# Project Views

- 형식: Table
- 용도: 전체 포트폴리오 확인

## Execution
- 형식: Board
- 컬럼: Backlog / Ready / In Progress / Validating / Blocked / Done

## Roadmap
- 형식: Roadmap
- 필드: Start Date / Target Date

## Bench Dashboard
- 형식: Table
- 필터: Type = benchmark

## Demo Readiness
- 형식: Table
- 필터: Demo Ready != No

## Blockers & ADRs
- 형식: Table
- 필터: Status = Blocked OR ADR Needed = Yes
